# 读读读

一个阅读小说的网站 (模仿书趣阁)


## 前台功能
- 展示分类下的所有小说
- 展示某小说的详情页
- 某小说的章节页面
- 排行榜
- 已读章节的状态
- 搜索小说, 作者, 章节

## 后台功能
- 管理主页面的极力推荐
- 管理主页面的强力推荐
- 小说列表
- 小说状态修改(已完结, 已删除)
- 新增小说
- 编辑小说
    - 基础信息修改
    - 添加章节
    - 删除章节
- 新增小说分类
- 系统设置
